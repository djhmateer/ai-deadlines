# Red:4 Marketplace API

A Node.js marketplace API project.

## Getting Started

### Prerequisites

- Node.js
- pnpm (v10.23.0)

### Installation

Install dependencies:

```bash
make install
# or
pnpm install
```

### Running the Application

Start the application:

```bash
make start
```

### Testing

Run the test suite:

```bash
make test
```

### Cleanup

Remove build artifacts:

```bash
make clean
```

## Development

This project uses `pnpm` as the package manager. All dependencies are managed through `package.json`.
